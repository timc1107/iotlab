# IoT Protocol Labs – MQTT & CoAP
Labs designed for students to learn and study how to use IoT application protocols like CoAP and MQTT 

## Objectives
- Understand MQTT publish/subscribe
- Understand CoAP REST over UDP
- Compare protocols for AIoT systems
- Build a real gateway translating CoAP → MQTT

## Prerequisites
- Python 3.9+
- Linux or WSL recommended
- Basic networking knowledge
